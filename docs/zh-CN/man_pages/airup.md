# 名称
airup - Airup的用户接口

# 大纲
**airup** 子命令 [参数 ...]

# 描述
**airup** 和 **airupd** 交互以管理和探视系统状态。

## 子命令

**start**        启动服务。

**stop**         停止服务。

**reload**       重载服务。

**restart**      重启服务。

**query**        查询系统信息。

**self-reload**  重载 **airupd** 服务自身。

**reboot**       重启、关闭或挂起系统。

**edit**         编辑 Airup 文件。

**enable**       启用一个单元。

**disable**      禁用一个单元。

**debug**        调试 Airup。

**help**         打印 **airup** 或指定子命令的帮助信息。
